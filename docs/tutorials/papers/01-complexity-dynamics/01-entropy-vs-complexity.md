# 01 熵是什么？熵和复杂度有什么区别？

问下大家，你们有没有思考过这样一个问题：**为什么宇宙从简单开始，变得越来越复杂，最后又回归简单？**

我刚学热力学的时候，也被这个问题困扰了很久。熵增原理告诉我们，系统的熵总是单调增加的，从有序走向无序。但看看我们周围——生命、文明、科技——这些复杂结构是怎么出现的？

直到我读到 Scott Aaronson 的文章，才发现卧槽，原来这里藏着一个深刻的理论问题！

## 熵：混乱度的度量

首先，让我们回顾一下熵的概念。

### 香农熵

在信息论中，**香农熵**（Shannon Entropy）度量的是一个随机变量的不确定性：

$$H(X) = -\sum_{i} p(x_i) \log_2 p(x_i)$$

举个栗子 🌰：
- 抛一枚均匀硬币，熵是 1 bit（两种可能，每种概率 0.5）
- 抛一枚有偏硬币（正面概率 0.9），熵小于 1 bit（结果更可预测）
- 抛一枚总是正面的硬币，熵是 0（没有不确定性）

### 热力学熵

在物理学中，**玻尔兹曼熵**定义为：

$$S = k_B \ln W$$

其中 $W$ 是系统的微观状态数。熵越高，系统可能的微观配置越多，看起来越"混乱"。

**关键洞察**：熵度量的是"可能性空间"的大小，而不是"有趣程度"。

## 复杂度：不只是混乱

现在问题来了：**均匀随机字符串的熵很高，但它有趣吗？**

想象一下这两个字符串：

```
字符串 A: 00000000000000000000  (20个0)
字符串 B: 10110101101011010110  (看起来随机)
字符串 C: 01010101010101010101  (简单模式)
```

- **A 的熵**：很低（几乎确定是0）
- **B 的熵**：很高（难以预测）
- **C 的熵**：很低（简单重复模式）

但从"复杂度"或"有趣性"的角度看：
- A 太简单了（全是0）
- B 虽然是随机的，但也没有结构
- C 有简单的结构，但太规律了

**真正的复杂度，似乎存在于"既不太简单，也不太随机"的中间地带！**

## 咖啡杯的例子

Sean Carroll 在演讲中用一个绝妙的例子说明这个问题：

![](https://scottaaronson.blog/coffee-small.jpg)

**左图**：牛奶和咖啡刚刚倒入，界限分明，熵低，复杂度低
**中图**：牛奶和咖啡正在混合，形成美丽的分形结构，熵中等，**复杂度最高**
**右图**：完全混合，均匀分布，熵最高，复杂度低

看到没？复杂度随时间的变化是：**低 → 高 → 低**，而熵是：**低 → 高 → 更高**（单调递增）！

## 为什么这很重要？

这个问题之所以重要，是因为：

1. **宇宙演化**：大爆炸初期简单 → 现在复杂（星系、生命） → 未来热寂（简单）
2. **生命起源**：从简单分子到复杂生命再到...？
3. **机器学习**：模型复杂度与泛化能力的关系
4. **信息压缩**：什么结构是可压缩的，什么不是？

## 核心挑战

Scott Aaronson 指出，要解释这个现象，我们需要解决两个挑战：

**挑战 1**：给出"复杂度"的严格数学定义

**挑战 2**：证明按照这个定义，复杂度确实会在中间时刻达到最大

这就引出了我们接下来要学习的工具：**Kolmogorov 复杂度** 和 **Sophistication**。

## 小结

问下大家，现在明白熵和复杂度的区别了吗？

| 特性 | 熵 (Entropy) | 复杂度 (Complexity) |
|------|-------------|-------------------|
| 简单系统 | 低 | 低 |
| 随机系统 | 高 | 低 |
| 复杂结构 | 中等 | **高** |
| 随时间变化 | 单调递增 | 先增后减 |

**关键洞察**：真正有趣的结构，既不是完全有序的（太简单），也不是完全随机的（太混乱），而是处于两者之间！
