# Sutskever 30 论文纯 NumPy 实现教程

## 项目简介

本教程系列是 Sutskever 30 篇经典深度学习论文的纯 NumPy 实现指南，旨在帮助开发者深入理解深度学习的核心机制。

### 核心特点

- **纯 NumPy 实现**：所有算法从零构建，无框架依赖
- **教育导向**：详细注释、数学公式、形状追踪
- **实战演练**：每个实现都配有可运行的示例代码
- **循序渐进**：从基础到进阶，系统掌握核心概念

### 目标读者

- 希望深入理解深度学习原理的开发者
- 想要摆脱框架束缚，掌握底层实现的研究者
- 准备面试，需要巩固核心知识的工程师

## 如何使用本教程

1. **克隆项目**
```bash
git clone <repository-url>
cd sutskever-30-implementations
```

2. **运行 Jupyter Notebook**
```bash
jupyter notebook
```

3. **跟随教程学习**
每个章节包含理论讲解和代码实现，建议按顺序学习。

## 代码风格

本项目遵循统一的代码风格：
