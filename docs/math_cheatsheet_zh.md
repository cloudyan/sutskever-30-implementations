## 2. 📐 张量形状：深度学习的"语言"

### 什么是张量？
- **标量**：一个数，如 `3.14`（0维张量）
- **向量**：一列数，如 `[1, 2, 3]`（1维张量）
- **矩阵**：二维数组，如 `[[1,2], [3,4]]`（2维张量）
- **张量**：三维或更高维数组（3维+张量）

### 深度学习中的常见形状
```
(batch_size, seq_len, hidden_size)
    ↓         ↓          ↓
  批次大小  序列长度   隐藏层维度
```

#### 实际例子：
- **图像**：`(32, 3, 224, 224)` = 32张图片 × 3通道 × 高224 × 宽224
- **文本**：`(16, 50, 768)` = 16个句子 × 50个词 × 每个词768维向量
- **音频**：`(8, 1000, 128)` = 8段音频 × 1000时间点 × 128个特征

### 形状追踪技巧
在代码注释中，你会看到：
```python
# x: (batch, seq_len, input_size)   输入
# W: (hidden_size, input_size)      权重
# b: (hidden_size,)                 偏置
# output: (batch, seq_len, hidden_size)  输出
```

**记住**：深度学习80%的bug都是形状不匹配！

## 4. 🧪 最小数值例子：看懂公式的第一步

### 例1：矩阵乘法
**公式**：C = A × B

**最小例子**：
```
A = [[1, 2],   B = [[5, 6],   C = [[1×5+2×7, 1×6+2×8],
     [3, 4]]        [7, 8]]        [3×5+4×7, 3×6+4×8]]
                = [[19, 22],
                   [43, 50]]
```

**代码**：
```python
A = np.array([[1,2],[3,4]])
B = np.array([[5,6],[7,8]])
C = A @ B  # 或 np.matmul(A, B)
```

### 例2：Softmax函数
**公式**：softmax(x_i) = exp(x_i) / ∑ exp(x_j)

**最小例子**：
```
x = [2.0, 1.0, 0.1]
exp(x) = [7.389, 2.718, 1.105]
sum = 7.389 + 2.718 + 1.105 = 11.212
softmax = [7.389/11.212, 2.718/11.212, 1.105/11.212]
         = [0.659, 0.242, 0.099]
```

**代码**：
```python
def softmax(x):
    exp_x = np.exp(x - np.max(x))  # 减去最大值防止溢出
    return exp_x / np.sum(exp_x)
```

## 5. 📊 线性代数基础（必须懂的3个概念）

### 1. 矩阵乘法
**核心思想**：行和列的点积
```
C[i,j] = A的第i行 · B的第j列
```

**形状规则**：
```
A: (m, n)   B: (n, p)   →   C: (m, p)
```

### 2. 转置（Transpose）
**核心思想**：行列互换
```
A = [[1, 2],   A.T = [[1, 3],
     [3, 4]]         [2, 4]]
```

**代码**：`A.T` 或 `np.transpose(A)`

### 3. 广播（Broadcasting）
**核心思想**：自动扩展小数组来匹配大数组

**例子**：
```python
A = np.array([[1,2,3],   # 形状 (2, 3)
              [4,5,6]])
B = np.array([10,20,30]) # 形状 (3,)

# B自动扩展为 [[10,20,30], [10,20,30]]
result = A + B  # 形状 (2, 3)
```

## 6. 📈 微积分基础（必须懂的3个概念）

### 1. 导数（Derivative）
**核心思想**：变化率
```
f(x) = x²
f'(x) = 2x  # 在x=3时，变化率是6
```

**深度学习意义**：告诉参数应该往哪个方向调整

### 2. 链式法则（Chain Rule）
**核心思想**：复合函数的导数 = 各层导数的乘积
```
y = f(g(x))
dy/dx = (df/dg) × (dg/dx)
```

**深度学习意义**：反向传播的基础

### 3. 梯度（Gradient）
**核心思想**：多变量函数的导数向量
```
f(x,y) = x² + y²
∇f = [∂f/∂x, ∂f/∂y] = [2x, 2y]
```

**代码**：`np.gradient(f)`

## 7. 🎲 概率论基础（必须懂的3个概念）

### 1. 概率分布
**核心思想**：描述随机变量的可能取值和概率

**常见分布**：
- **均匀分布**：每个值概率相等
- **正态分布**：钟形曲线
- **伯努利分布**：0或1（二分类）

### 2. 期望值（Expectation）
**核心思想**：平均值
```
E[X] = ∑ x_i × P(x_i)
```

**代码**：`np.mean(x)`

### 3. 最大似然估计（MLE）
**核心思想**：选择使观测数据概率最大的参数

**例子**：抛硬币10次，7次正面，3次反面
- 最大似然的正面概率 = 7/10 = 0.7

## 8. 🔬 信息论基础（必须懂的3个概念）

### 1. 熵（Entropy）
**核心思想**：不确定性/信息量
```
H(X) = -∑ P(x) × log P(x)
```

**直观理解**：
- 确定性事件：熵=0（无信息）
- 完全随机：熵最大（信息最多）

### 2. KL散度（Kullback-Leibler Divergence）
**核心思想**：两个概率分布的差异
```
KL(P||Q) = ∑ P(x) × log(P(x)/Q(x))
```

**深度学习应用**：VAE的正则化项

### 3. 互信息（Mutual Information）
**核心思想**：两个变量的相关程度
```
I(X;Y) = H(X) - H(X|Y)
```
